# Citation Clerk Sim — Instructor Dashboard (static)

## What it is
A static, offline dashboard for aggregating exported JSON files from the Citation Clerk Simulation.
No server, no database, and nothing is uploaded anywhere.

## Use
1) Click "Add JSON files" or drag-and-drop student result files into the drop zone.
2) Click a row to view individual detail.
3) Export class summary as CSV.
4) Download a shareable snapshot HTML that embeds the currently loaded data.

## Privacy
All processing happens in your browser. Exported snapshot HTML embeds the data you loaded.
